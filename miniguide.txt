1. Зайти в папку "output", в файл teams.xml (открыть с помощью -> Блокнот) записать имена игроков обеих команд;
2. В папку stuff/1280_720_new поместить логотипы команд, переименовать в logo1 для первой и logo2 для второй (формат png);
3. В OBS добавить в источники браузер -> галочка "локальный файл" -> выбрать файл versus_1280_720_new.html -> задать ширина: 1280, высота: 140;
4. Также добавить браузер, сняв галочку "сделать источник видимым" -> выбираем easing_1280_720_new.html -> задать ширина: 1280, высота: 720;
5. Запустить ScoreBoard Assistant -> в поля player1 и player2 вписать имена команд 1 и 2 соответственно, в поле match меняем пишем четверть;
6. Во время игры меняем счет/четверть, после каждой смены жмем Save;
